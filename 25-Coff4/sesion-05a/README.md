VCV: VCV Rack, un software de síntesis modular virtual 

- VCO: Es un circuito electrónico que genera una señal de salida (sinusoidal, cuadrada o triangular) cuya frecuencia cambia de forma proporcional a una tensión continua de entrada

- LFO: Oscilador secundario que opera a frecuencias inaudibles, diseñado para modular parámetros de otros componentes en sintetizadores y efectos.

- Onda Seno (SIN): Visualización del sonido (en forma de oleaje)

- TRI: Seudo triangulas (cada una tiene diferentes maneras de expresar el sonido)

- SAW: En forma de sierra (Por lo que aparenta es más rapida)

- SQR (Square): Como su nombre indica, es en forma de cuadrado

Smith Trigger: ¿que es?

- tipo especial de circuito comparador

- convertir señales inestables o ruidosas en señales digitales limpias

¿Como funciona?

Primero definimos un voltaje

Si el voltaje es mayor al numero definido, se accionara un 1

Si el voltaje es menor se transmitira un 0

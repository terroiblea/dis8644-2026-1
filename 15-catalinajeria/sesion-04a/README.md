En esta clase se repasó el funcionamiento del chip 555 en modo astable y se introdujo el uso de simuladores de circuitos como herramienta para probar conexiones de forma digital antes de armarlas físicamente.

También se dieron indicaciones para un encargo práctico relacionado con el análisis de objetos electrónicos.

### Repaso chip 555 (modo astable)

Se reforzó el funcionamiento del chip 555 en modo astable:

- Genera una señal que oscila constantemente (sube y baja)  
- Permite crear ciclos repetitivos (como parpadeo o sonido continuo)  

Se revisaron las patas (pines) del chip:

- Pin 4 (Reset):  
  - Si está conectado a positivo → el circuito funciona normalmente  
  - Si no está conectado a positivo → no hay oscilación  

- Pin 8 y Pin 4:  
  - Ambos van conectados a positivo (VCC)  

- Pin 5:  
  - Puede usarse o no, dependiendo del circuito  

También se recordó que las líneas de la protoboard se llaman rieles.

### Astable vs Monostable

Se compararon los dos modos del chip 555:

- Astable (AS):  
  - Oscila constantemente  
  - No necesita intervención externa  

- Monostable (MS):  
  - Necesita un impulso para activarse  
  - Genera una sola señal  

### Simulación de circuitos

Se utilizó la herramienta **Falstad Circuit Simulator**:

- Permite simular circuitos de forma digital  
- Se pueden copiar y pegar códigos para generar circuitos automáticamente  
- Sirve para probar antes de armar en físico  

Esto ayuda a entender mejor cómo funcionan los circuitos sin necesidad de conectar componentes reales.

### Conceptos importantes

- Los pines del chip tienen funciones específicas  
- La conexión del reset define si el circuito funciona o no  
- Es importante entender la diferencia entre los tipos de señal (continua vs pulso)

### conceptos 
- VC = controlado por voltaje
- VCO = oscilador ue se puede controlar por voltaje 
- LFO = no se controla por voltaje, oscilador de baja frecuencia
- DAC = chips q transforman () a sonido
- SIN = onda comun, sube y baja sube y baja 
- TRI = onda mas recta, triangular
- SAW = onda sierra, mas violenta en el sonar, tiene mas info q las anteriores
- SQE = la que hemos trabajado nosotros, baja se mantiene y baja, es un cuadrado
- VCA = sube y baja el volumen
- NAND: se mantiene prendido solo si 0 1 o 1 0, no 0 0 o 1 1
- AND: se enciende si y solo si  1 1 y apaga con 0 0

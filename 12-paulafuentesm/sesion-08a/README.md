+ nos permitirá diseñar esquemas electrónicos y convertirlos en placas para mandar después a cerrar chine para las siguientes entregas

### Comandos Importantes: ( que se iban olvidando )
+ a : añadir componente
+ v : poner el valor de un componente
+ crl + c/v/x : tipicos comandos copiar, pegar, cortar
+ r : rotar
+ g : arrastar todo el componente con el cable
-----

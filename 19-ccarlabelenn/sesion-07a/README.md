## KiCad 

software libre para la automatización del diseño electronico, vamos a aprender a hacer circuitos con placas cubicadas, veremos como matias logro hacer todos los esquematicos, nos ocuparemos en aspectos de carcasa, muy bacan.

son tres procesos de diseños distintos :
+ dibujar el esquematico
+ layout de pcb, se dibuja cada cable y se vuelven planos
+ ver como se ven las placas en 3D
+ pcb manofacturing
